This is first in series of posts about blogging with LaTeX using TeX4ht. I’ve set it up as a showcase of TeX4ht features for TUG 2021.

In the following posts, I will tell more on how to set up an automatic blog publishing using Github Pages and Github Actions.

Overview of the process is following:

1.
Set up TeX4ht to produce files suitable for static site generators.
2.
Set up static site generator, like Jekyll or Hugo , to produce a web site.
