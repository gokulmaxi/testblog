This is first in series of posts about blogging with LaTeX using TeX4ht. I’ve set it up as a showcase of TeX4ht features for TUG 2021.

Overview of the set up is following:

  1. Use TeX4ht to produce files suitable for static site generators.
  2. Use static site generator, like Jekyll or Hugo , to produce a web site.
  3. Use Github Actions to automatically compile LaTeX files pushed to the repository, and recompile the web site.

In the following posts, I will tell more on how to set up an automatic blog publishing using Github Pages and Github Actions.
